# Full-Stack Task Manager (Python/FastAPI)

A streamlined, functional Task Manager application designed to showcase clean API design and responsive frontend logic. This project was built to satisfy the requirements of the Full Stack Developer Technical Assignment within a strict 2-hour window.

Key Features:
* Complete CRUD: Create, View, Toggle Completion, and Delete tasks.
* Bonus - Persistence: Tasks are saved to a local JSON file so they remain after a server restart.
* Bonus - Filtering: Toggle between "All", "Active", and "Completed" views.
* Bonus - Editing: Inline task title updates via the UI.
* Bonus - Testing: Includes a Python-based test suite to verify API integrity.
* Zero-Dependency Frontend: Pure HTML5/JavaScript for maximum speed and compatibility.
